## 🛠️ Requisitos Implementados
[cite_start]Según lo solicitado en el enunciado del trabajo[cite 5]

1.  [cite_start]Estructura de Carpetas Organización profesional en carpetas `css`, `js`, `views` e `images`[cite 76, 77].
2.  [cite_start]Página de Inicio (index.html) Contiene 4 secciones y carga dinámica de noticias desde un archivo externo `noticias.json` mediante la Fetch API[cite 12, 13, 14].
3.  [cite_start]Galería Dinámica Implementada en `galeria.html` utilizando JavaScript para la generación de elementos de imagen[cite 15, 16].
4.  [cite_start]Presupuesto Interactivo  Validación de datos de contacto con expresiones regulares (Nombre  15 caracteres, Apellidos  40, Teléfono de 9 dígitos)[cite 20, 21, 25, 26, 27].
     [cite_start]Cálculo automático del presupuesto final en tiempo real sin recarga de página[cite 29, 37, 38].
5.  [cite_start]Mapa Dinámico Ubicación de la empresa y funcionalidad de ruta integrada en `contacto.html` usando la librería Leaflet[cite 53, 54, 55].
6.  [cite_start]Estilo y Navegación Barra de navegación fija (stickyfixed), resaltado de la página activa mediante CSS y pie de página con información legal y redes sociales[cite 65, 66, 67, 68].

## 🚀 Instalación y Uso
1.  Clonar el repositorio o descargar el ZIP.
2.  Abrir el archivo `index.html` en cualquier navegador moderno.
3.  Para visualizar las noticias, se recomienda abrir el proyecto mediante un servidor local (como Live Server en VS Code) para evitar restricciones de CORS del navegador al leer el archivo JSON local.
